# Education Analyst Skill

## Objetivo

Analisar indicadores educacionais do World Bank Education Statistics e gerar insights utilizando IA.

## Entradas

- País
- Indicador
- Ranking
- Dados históricos

## Processamento

- Comparar países
- Identificar tendências
- Detectar crescimento ou queda
- Gerar ranking
- Produzir recomendações

## Saídas

- Ranking dos países
- Comparação entre países
- Insights executivos
- Recomendações para políticas públicas

## Ferramentas

- Python
- Pandas
- FastAPI
- Groq LLM
- n8n
